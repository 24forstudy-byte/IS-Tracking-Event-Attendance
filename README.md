# IS-Tracking-Event-Attendance

Система предназначена для учёта мероприятий и регистрации участников.

Цели ИС: 

− 	Хранение информации о мероприятиях и их характеристиках.

− 	Хранение и ведение базы участников и их контактных данных.

− 	Регистрация заявок на конкретные мероприятия.

− 	Формирование списков зарегистрированных участников.

− 	Отчетность по участникам и мероприятиям.

Инструкция по установке и запуску

Как установить и запустить:

1. Установите Python с официального сайта python.org.
2. Склонируйте репозиторий в терминале с помощью команды:
   
   git clone https://github.com/24forstudy-byte/IS-Tracking-Event-Attendance.git
   
3. Перейдите в папку проекта:

   cd IS-Tracking-Event-Attendance

4. Выполните команду:
   
   python main.py
   
  После запуска откроется консольное меню управления системой.
